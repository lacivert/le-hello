Gives you a `Hello` string with the name you sent, or `you` if not a name specified.

## Install
`npm i le-hello` 

## Usage

``` js
const hello = require("le-hello");

console.log(hello("lacivert"));
// Hello, lacivert!

// or you can use without a param
console.log(hello());
// Hello, you!
```
